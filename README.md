# Bozas Quick Entry — Bridge Repository

This repository is a communication channel between Claude (claude.ai) and Claude Code working on the **Bozas Quick Entry** project.

## Purpose

- Claude.ai writes briefings and instructions here for Claude Code to read
- Claude Code posts questions and feedback here for Claude.ai to respond to
- This is **not** the source code repository — that one is private

## Contents

| File | Purpose |
|------|---------|
| `BRIEFING.md` | Full project context: architecture, decisions, tech stack |
| `current-task.md` | What Claude Code should be working on right now |
| `questions.md` | Questions from Claude Code to Claude.ai |

## Project

Voice-driven quick entry tool for Pylon ERP. Mobile-first, Greek language, integrated with Pylon Connectivity API.
